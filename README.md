# VBA Inventor: RKM рамка + штамп (первая рабочая версия)

## Что делает проект
Этот репозиторий содержит **только исходники VBA** для Autodesk Inventor, которые создают и применяют:
- отдельный `BorderDefinition` (рамка A3 landscape + левая вертикальная полоса согласования);
- отдельный `TitleBlockDefinition` (штамп);
- реальные поля `Prompted Entry` в штампе через `<Prompt>...</Prompt>` и `Sheet.AddTitleBlock(..., PromptStrings)`.

## Текущий scope
В этой фазе реализованы только:
1. рамка;
2. штамп;
3. порядок prompt-полей;
4. публичные макросы запуска.

## Что намеренно **вне scope** сейчас
- верхние таблицы «Ведомость...»;
- нижний левый технический текстовый блок;
- экспорт в `.idw/.dwg`, сборки, ZIP, скриншоты.

## Структура
- `src/RKM_EntryPoints.bas` — публичные макросы запуска.
- `src/RKM_FrameBorder.bas` — создание/применение BorderDefinition.
- `src/RKM_TitleBlockPrompted.bas` — создание/применение TitleBlockDefinition с prompt-полями.
- `src/RKM_Utils.bas` — общие функции, безопасная обработка имен/ресурсов, дефолтные значения prompt.
- `docs/*.md` — спецификация, порядок полей, ручной чек-лист, ограничения.

## Импорт модулей в Inventor VBA
1. Откройте Inventor.
2. Откройте VBA Editor (обычно `Alt+F11`).
3. `File -> Import File...` и импортируйте все `.bas` из `src/`.
4. Выполните `Debug -> Compile VBAProject`.

### Важно про `Attribute VB_Name`
- При **Import File** строку `Attribute VB_Name = "..."` оставляйте как есть.
- При **ручной вставке кода** в окно модуля сначала удалите строку `Attribute VB_Name`, затем вставляйте остальной код.

## Какой макрос запускать первым
Рекомендуемый первый запуск:
- `RKM_CreateNewDrawingAndApplyFrameAndTitleBlock`

Остальные публичные макросы:
- `RKM_ApplyFrameAndTitleBlockToActiveSheet`
- `RKM_RebuildDefinitionsInActiveDrawing`

## Стратегия по повторному запуску и конфликтам имен
- Базовые имена ресурсов: `RKM_A3_BORDER`, `RKM_A3_TITLEBLOCK`.
- Если definition с таким именем **существует и не используется**, он удаляется и пересоздается.
- Если definition **уже используется** (`IsReferenced=True`), создается новая версия с суффиксом `_v2`, `_v3`, ...
- Перед вставкой на лист текущие `Sheet.Border` и `Sheet.TitleBlock` удаляются безопасно.

## Кодировка и языковые caveats
- В `.bas` использованы русские строки (подписи и тексты prompt), так как это целевой UX.
- В некоторых конфигурациях VBA импорт чувствителен к кодировке; при искажении текста:
  1. проверьте путь импорта через `Import File`;
  2. не копируйте через промежуточные редакторы, меняющие кодировку;
  3. ориентируйтесь на `docs/prompt-fields.md` для восстановления подписей.
