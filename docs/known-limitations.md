# Известные ограничения

1. Это первая исходная версия и она не включает:
   - верхние таблицы «Ведомость...»;
   - нижний левый технический абзац.

2. Геометрия и типографика близки к целевому виду, но не зафиксированы как финальный корпоративный шаблон.

3. Автоматические тесты Inventor API в CI отсутствуют: проверка выполняется вручную в установленном Autodesk Inventor.

4. В `.bas` присутствуют русские строки (prompt-метки и подписи). В редких окружениях возможны проблемы кодировки при неправильном импорте.
   Рекомендуется импортировать модули через `File -> Import File...` в VBA Editor.

5. Использование `Sheet.ChangeSize(kA3DrawingSheetSize, kLandscapePageOrientation)` оставлено с мягкой обработкой ошибок (On Error), т.к. поведение зависит от версии Inventor/шаблона документа.

6. Для уже используемых (`IsReferenced`) definitions применяется стратегия версионирования имени (`_v2`, `_v3`, ...), а не принудительная замена ссылок во всех листах.
