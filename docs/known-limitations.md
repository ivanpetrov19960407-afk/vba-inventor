# Известные ограничения

1. Текущий scope по-прежнему ограничен:
   - рамкой листа;
   - основной надписью СПДС форма 3.
   Верхние «Ведомость...» таблицы и нижний левый технический абзац не реализуются в этом репозитории.

2. Толщины линий и точные шрифты формы 3 не параметризованы отдельно (используются настройки документа/стиля Inventor).

3. Автоматические тесты Inventor API в CI отсутствуют: проверка выполняется вручную в установленном Autodesk Inventor.

4. Для кириллицы в `.bas` принята стратегия **B**: строки собираются через `ChrW(...)` (`RuText`).
   Это снижает риск проблем кодировки при импорте модулей.

5. Если активен режим редактирования эскиза/ресурса (`ActiveEditObject` не `Nothing`), макрос останавливается с сообщением.
   Автоматический выход из edit-режима не выполняется намеренно.
