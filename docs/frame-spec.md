# Спецификация рамки и штампа СПДС (ГОСТ Р 21.101-2020)

## Нормативная привязка
- П.5.2.1: основная надпись размещается в правом нижнем углу листа; применяются формы 3–6 (приложение Ж).
- П.5.2.2: для листов основных комплектов рабочих чертежей и графической части применяется **форма 3**.
- П.5.2.7: размеры рамок и их расположение принимаются по приложению И.
- Формат A3: `420 x 297 мм` (ГОСТ 2.301), ориентация landscape.

## Почему архитектура разделена на BorderDefinition + TitleBlockDefinition
Использован штатный подход Inventor:
- рамка как отдельный ресурс через `DrawingDocument.BorderDefinitions` и `Sheet.AddBorder`;
- штамп как отдельный ресурс через `DrawingDocument.TitleBlockDefinitions` и `Sheet.AddTitleBlock`.

## SPDS baseline (source of truth)
- Лист A3 landscape: `420 x 297 мм` (`42.0 x 29.7 см` в API).
- Внутренняя рамка рабочего поля: отступ `20 мм` слева и `5 мм` с остальных сторон (приложение И).
- Основная надпись СПДС форма 3: `185 x 55 мм` (приложение Ж).
- Положение штампа: правый нижний угол внутренней рамки.

## Что реализовано фактически
### BorderDefinition (`RKM_SPDS_A3_BORDER`)
- Один прямоугольник внутренней рамки по отступам `20/5 мм`.

### TitleBlockDefinition (`RKM_SPDS_A3_FORM3_TITLEBLOCK`)
- Прямоугольник основной надписи `185 x 55 мм`.
- Внутренняя сетка базовой структуры формы 3.
- Статические подписи + prompted-поля (`<Prompt>...</Prompt>`) для заполняемых значений.
