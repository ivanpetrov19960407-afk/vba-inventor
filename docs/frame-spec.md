# Спецификация рамки и штампа (первая итерация)

## Почему архитектура разделена на BorderDefinition + TitleBlockDefinition
Использован штатный подход Inventor:
- рамка как отдельный ресурс через `DrawingDocument.BorderDefinitions` и `Sheet.AddBorder`;
- штамп как отдельный ресурс через `DrawingDocument.TitleBlockDefinitions` и `Sheet.AddTitleBlock`.

## Геометрический baseline
- Лист A3 landscape: `420 x 297 мм` (`42.0 x 29.7 см` в API).
- Внутренняя рамка рабочего поля: отступ `20 мм` слева и `5 мм` с остальных сторон.
- Основная надпись (форма 1): `185 x 55 мм`.
- Положение штампа: правый нижний угол внутренней рамки.

## Что реализовано фактически
### BorderDefinition
- Один прямоугольник внутренней рамки по отступам `20/5 мм`.

### TitleBlockDefinition
- Прямоугольник основной надписи `185 x 55 мм`.
- Внутренняя сетка штампа (упрощённая форма 1).
- Статические подписи в штампе.

## Сознательные упрощения первой версии
- Толщины линий и точные шрифты не параметризованы отдельно.
- Верхние «Ведомость...» таблицы и нижний левый технический абзац не реализованы (вне текущего этапа).
