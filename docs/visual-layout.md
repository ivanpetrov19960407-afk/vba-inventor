# Визуальная раскладка видов (A3, СПДС)

Макрос альбома формирует **3 ортогональных вида** на каждом листе `ALB_*`:
- фронтальный (base);
- верхний (projected top);
- боковой (projected side).

## Зоны листа
- `safeRect`: внутренняя область рамки (20 мм слева, 5 мм с остальных сторон).
- `blockedRect`: зона штампа в правом нижнем углу (`185 x 55 мм`) — виды не должны пересекать её.

```text
+--------------------------------------------------------------+
|                         TOP VIEW RECT                        |
|                                                              |
|-------------------------------+------------------------------|
|         FRONT VIEW RECT       |        SIDE VIEW RECT        |
|                               |                              |
|                               |                              |
|                               |                              |
|                               |                              |
|-------------------------------+------------------------------|
|                               blocked title block            |
+--------------------------------------------------------------+
```

## Учёт стандарта проекций (ISO/ANSI)
- Определяется через `ActiveStandardStyle.FirstAngleProjection`.
- Если `True` (ISO, first-angle):
  - верхний вид создаётся как `kAboveViewOrientation`;
  - боковой создаётся как `kLeftViewOrientation` (подпись «Вид слева»).
- Если `False` (ANSI, third-angle):
  - верхний вид создаётся как `kAboveViewOrientation`;
  - боковой создаётся как `kRightViewOrientation` (подпись «Вид справа»).

## Настраиваемые константы
В `src/RKM_IdwAlbum.bas`:
- `TOP_ROW_RATIO` — доля высоты под верхнюю полосу (top view).
- `SIDE_COL_RATIO` — доля ширины под боковую колонку (side view).
- `GAP_MM` — зазор между зонами видов.
- `LAYOUT_PAD_MM` — внутренний отступ от safeRect.
- `LABEL_OFFSET_MM`, `LABEL_MARGIN_MM` — сдвиг подписей видов.

## Подписи видов
Для всех трёх видов включается `ShowLabel = True` и добавляется русский заголовок над системной подписью:
- «Вид спереди»
- «Вид сверху»
- «Вид слева» / «Вид справа» (в зависимости от стандарта)
